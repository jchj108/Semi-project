    Chrome 위치정보 설정하기

    1. 크롬 브라우저 오른쪽 상단에서 '더보기' > '설정'을 클릭합니다.
    2. 생성된 창에서 하단의 '고급'을 클릭합니다.
    3. '개인정보 및 보안'에서 콘텐츠 설정을 클릭합니다.
    4. '위치'를 클릭합니다.
    5. '엑세스하기 전에 확인'을 사용으로 변경합니다.
    그 외 'chrome://settings/content/location'에 접속하셔서 '5번'의 절차를 진행 합니다.

    Safari 위치정보 설정하기

    1. 브라우저 오른쪽 상단에서 '설정’ > '기본설정’을 클릭합니다.
    2. 기본설정 팝업창에서 '개인정보’ 아이콘을 클릭합니다.
    3. '위치 서비스 웹 사이트 접근제한’에서 '하루에 한번씩 각 웹 사이트에 대해 입력요청’ 항목을 선택합니다.
    4. 페이지 새로고침을 통해 위치정보 요청 팝업에서 '허용’을 클릭 합니다.
    당일 이미 위치정보 요청을 거절하였다면 브라우저 오른쪽 상단에서 '설정’ > 'safari재설정’ > '모든위치경고 재설정’을 체크한 뒤 > 재설정 버튼을 클릭 합니다.

    Internet Exploer 11 위치정보 설정하기

    1. 브라우저 오른쪽 상단에서 '설정’ > '인터넷옵션’을 클릭합니다.
    2. 인터넷 옵션 창에서 '개인정보’ 탭을 클릭 합니다.
    3. '위치’에서 '웹 사이트에서 사용자의 위치 정보 요청을 허용안함’을 체크 해제합니다.

    그래도 위치정보가 노출되지 않는다면!
    A. Windows 10을 사용하는 경우 PC의 '시작’ 버튼을 클릭 한 뒤 노출되는 시작메뉴의 '설정’ 버튼을 클릭합니다.
    B. 설정창에서 '개인정보’ > '위치’ 를 클릭 합니다.
    C. '이 장치의 위치 켬’에서 '변경’ > '이 장치의 위치’> '켬’ 버튼을 클릭 합니다.

    기타 브라우저

    현재 브라우저 위치정보가 꺼져 있습니다.
    현위치를 반영하기 위해서는 이용하시는 브라우저의 위치정보를 '활성화' 시켜주시기 바랍니다.
    위치정보 활성화에 대한 자세한 안내는 이용하는 브라우저 제공사의 고객센터에 확인 또는 문의하시기 바랍니다.
